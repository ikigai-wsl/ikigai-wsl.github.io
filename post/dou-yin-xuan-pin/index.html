选词指标
热度指数 （一定程度上可以代表搜索人数，低于500可以不考虑） ⬆️
点击率 （点击次数 / 访问次数，低于5%不考虑）⬆️
成交转化率 （支付买家数 / 搜索人数，5% - 60%）⬆️
关联商品指数 （搜索关键字的在线商品数，小于20大概率是违规词，5 - 2000）⬇️
竞争指数 ⬇️

蓝海指数
热度指数 * 点击率 = 点击人数
点击人数 * 成交转化率 = 付款人数
付款人数 / 关联商品指数 = 平均每个商品付款人数 / 统计时间天数 = 平均每天每个商品付款人数 （数值较下，不方便对比一般再乘上1000） * 1000

通过上述公式可以得出蓝海指数
